SkinSense is a cloud-based skincare ingredient analysis platform.

Users can search for skincare products and determine whether they are suitable for their skin type based on ingredients.

## Tech Stack

Frontend
- React
- TypeScript

Backend
- Python
- FastAPI

Cloud
- Azure Static Web Apps
- Azure App Service
- Azure Blob Storage

## Project Structure

frontend → React UI  
backend → FastAPI API  
dataset → ingredient datasets  
docs → architecture documentation
